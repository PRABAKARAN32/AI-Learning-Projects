# AI Agent with LangChain

A simple AI-powered application built using **LangChain** and **LCEL (LangChain Expression Language)**. This project allows users to interact with an AI model by assigning a role, topic, and tone, generating detailed and context-aware explanations.

## Features

- Dynamic role-based AI responses
- Customizable explanation tone
- Interactive command-line interface
- Built with LangChain and LCEL

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Enter:

- A role (e.g., Senior Software Engineer)
- A topic to explain
- A preferred tone (e.g., Professional, Friendly)

The AI will generate a detailed explanation based on your inputs.

- Python
- LangChain
